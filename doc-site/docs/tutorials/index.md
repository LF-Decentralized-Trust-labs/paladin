## Pre-requisites

* git
* [NodeJS 20.x or newer](https://nodejs.org/en/download/package-manager) installed

Clone the Paladin repository to access the examples:

```shell
git clone https://github.com/LF-Decentralized-Trust-labs/paladin.git
```
